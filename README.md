# movinglight-js

ムービングライトをnodejsで制御する

## 概要
scenesフォルダに、jsで記述されたスクリプトを書く。

oscサーバーに発火したいsceneを知らせると、sceneスクリプトに応じて、oscまたはdmx信号を送付して、ムービングライトを制御する。
